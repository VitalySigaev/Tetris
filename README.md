### Задание:
Игра «Тетрис» <br/>
Разработайте классическую игру «Тетрис». <br/>
Реализуйте механику падающих фигур, их повороты и перемещения. <br/>
Добавьте систему счета и проверку на конец игры при достижении верхней границы.
